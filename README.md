# GST JSON Generator Pro 🚀

A professional desktop application built with **Python + PySide6** to generate **GST Portal compatible GSTR-1 JSON files** from marketplace reports like **Meesho, Flipkart, and Amazon**.

---

## ✨ Features

* ✅ Generate GST compliant **GSTR-1 JSON**
* ✅ Supports:

  * Meesho
  * Flipkart
  * Amazon
* ✅ Auto detect file structure
* ✅ Excel / CSV support
* ✅ State-wise tax summary
* ✅ Dynamic totals calculation
* ✅ Invoice series generation
* ✅ SHA256 hash generation
* ✅ Professional GUI (PySide6)
* ✅ Validation before export

## ⚙️ Installation

### 1️⃣ Clone Repo

```bash
git clone https://github.com/YOUR_USERNAME/gst-json-generator-pro.git
cd gst-json-generator-pro
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### 3️⃣ Activate

### Linux / Mac

```bash
source venv/bin/activate
```

### Windows

```bash
venv\Scripts\activate
```

### 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Project

```bash
python main.py
```
